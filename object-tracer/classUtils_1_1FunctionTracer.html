Detailed Description

Class whose constructor/destructor pair are used to mark entering and leaving a function body block. You do this by defining an automatic variable of CodeTracker type at the beginning of the body. The constructor triggers an appropriate message.The destructor for the variable gets called when leaving the function body and triggers the appropriate message. The messages are logged through the TrackerOutput.

Detailed Description

Class whose constructor/destructor pair are used to mark entering and leaving a code block. You do this by defining an automatic variable of BlockTracer type at the beginning of the block. The constructor triggers an appropriate message.The destructor for the variable gets called when leaving the block and triggers the appropriate message. The messages are logged through the tarcer log.

Detailed Description

Class whose constructor/destructor pair are used to mark entering and leaving a member (function) body block. You do this by defining an automatic variable of MemberTracer at the beginning of the member body. The constructor triggers an appropriate message.The destructor for the variable gets called when leaving the member function body and triggers the appropriate message. The messages are logged to the tracer_log.
